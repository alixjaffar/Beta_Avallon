# Travel Blog

This is a blog website for a travel blogger, built using Next.js 14+ with the App Router, TypeScript, and Tailwind CSS.

## Features

- Responsive and modern design
- Photo gallery for showcasing travel photos
- Social media links for the blogger
- Frequently Asked Questions (FAQ) section
- Proper SEO meta tags

## Technologies Used

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- @shadcn/ui components
- Radix UI components (Accordion, Dialog, Dropdown Menu, Toast)
