# Auto Detailing Website

This is a professional, modern, and fully functional auto detailing website built with Next.js 14+ and App Router, TypeScript, and Tailwind CSS.

## Features

- **Next.js 14+ with App Router**: The website is built using the latest version of Next.js, which includes the new App Router for improved performance and developer experience.
- **TypeScript**: The project uses TypeScript for type safety and improved development experience.
- **Tailwind CSS**: The website is styled using Tailwind CSS, a utility-first CSS framework that provides a modern and responsive design.
- **shadcn/ui**: The project utilizes the shadcn/ui library for reusable UI components, providing a consistent and professional look and feel.
- **Responsive Design**: The website is designed to be mobile-friendly and adapts to different screen sizes.
- **SEO**: The website includes proper SEO meta tags to ensure optimal visibility in search engines.
- **Error Handling and Loading States**: The website includes proper error handling and loading states to provide a seamless user experience.
