## 🔬 What is Kirin? (In-House Prompt Engineering System)

**Kirin** is Avallon's proprietary prompt engineering system—think of it as an expert prompt engineer we built in-house that knows exactly how to talk to AI models to get production-ready websites.

### How Kirin Works:

**1. Intelligent Request Analysis**
- Kirin analyzes every user request to understand intent:
  - Is this a style change? (colors, fonts, spacing)
  - A layout modification? (structure, positioning)
  - Content addition? (new sections, features)
  - Interactive feature? (carousels, modals, forms)
  - Simple text edit? (just changing words)

**2. Context-Aware Prompt Building**
- Kirin builds sophisticated prompts tailored to each scenario:
  - **New Websites**: Multi-page generation with navigation, SEO, accessibility
  - **Modifications**: Surgical edits that preserve existing design
  - **Cloning**: Advanced website replication with SiteMirror integration
  - **Iterations**: Chat history context for continuous improvements

**3. Quality Bar Enforcement**
- Kirin enforces "Lovable-level" quality standards:
  - Distinct visual concepts (no generic layouts)
  - Layered backgrounds, gradients, section dividers
  - Social proof sections and narrative flow
  - Cohesive design systems with CSS variables
  - Lightweight motion and micro-interactions
  - WCAG 2.1 AA accessibility compliance
  - Performance optimization (Lighthouse 90+)

**4. Production-Ready Output**
- Kirin ensures every generation includes:
  - Clean, semantic HTML
  - Optimized CSS with variables
  - Vanilla JavaScript (no frameworks required)
  - Responsive design (mobile-first)
  - SEO optimization (meta tags, structured data)
  - Cross-browser compatibility

### Why Kirin Matters:

**With Kirin** (our system):
